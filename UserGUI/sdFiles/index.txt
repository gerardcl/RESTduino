					<p>
						Select an Arduino in the network writing its IP. Click "Connect" for update its configuration. You can configure the input
						ports clicking on the buttons and selecting the type of input.
					</p>
					<p>
						In case of leaving an input or output empty, select the empty option.
					</p>
					<p>
						<span class="label label-warning">Warning</span> The ouputs "O0" and "O1" are reserved for the Network and the File storage controllers. They
						will be always on high level (On).
					</p>
					<h4>Port Types</h4>
					<p><span class="label label-info">Raw</span> This type is the Raw value read by the Arduino's ADC. The value is represented as a 10 bit unsigned integer, from 0 to 1023.</p>
					<p><span class="label btn-danger">Temperature</span> This type is the temperature read from a thermistor connected in the analogical input port. The value is represented as a Celsius degrees with one figure.</p>
					<p><span class="label label-success">Potentiometer</span> This type is the potentiometer read from a potentiometer connected in the analogical input port. The value is represented as a decimal percentage with one figure.</p>
					<p><span class="label label-warning">Light</span> This type is the light read from a LDR connected in the analogical input port. The value is represented with the Raw value of the ADC.</p>
					<p><span class="label btn-primary">Logical</span> This type is the logic read from a switch connected in the analogical input port. The value is represented with using a logic states: True (On) or False (Off).</p>
